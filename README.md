# Tarea Ayudantía 3: CRUD de Alumnos

Este programa consiste en un sistema **CRUD** (Create, Read, Update, Delete) simulado en memoria utilizando JavaScript. La aplicación permite gestionar un listado de alumnos para la asignatura de **Metodologías de Desarrollo**, operando a través de una interfaz de consola interactiva.

## 📋 Contexto
El objetivo de esta tarea es demostrar el manejo de arreglos (`arrays`), objetos y lógica de programación en JavaScript (Node.js), simulando el comportamiento de las operaciones de una base de datos pero en memoria volátil.

## 🚀 Funcionalidades
El sistema incluye las siguientes operaciones principales:

* **Listar Alumnos:** Muestra todos los registros actuales en formato de tabla.
* **Crear Alumno:** Permite registrar un nuevo estudiante con un ID único autoincremental.
* **Buscar por ID:** Recupera la información de un alumno específico mediante su identificador.
* **Editar Alumno:** Actualiza de forma selectiva los campos (nombre, email o sección) de un registro existente.
* **Eliminar Alumno:** Remueve a un estudiante del arreglo principal.
* **Buscar por Sección:** (Desafío Extra) Filtra y despliega a los alumnos que pertenecen a una sección específica.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** JavaScript
* **Entorno de ejecución:** Node.js
* **Interfaz:** Módulo nativo `readline` para interacción por consola.
